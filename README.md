SPFMUI

Turn your handwriting into a sitelen pona font!

A graphical user interface for [SP Font Maker](https://github.com/KelseyHigham/sp-font-maker) by Waso Keli.

## Quick Start

### Mac
1. Double-click `setup_mac.command` (one time)
2. Double-click `launch_mac.command` (every time)

### Windows
1. Double-click `setup_windows.bat` (one time)
2. Double-click `launch_windows.bat` (every time)

## Requirements

- [FontForge](https://fontforge.org/en-US/downloads/) (must be installed separately)
- sp-font-maker from Waso Keli

## What it does

1. Import your scanned template
2. Map custom words to slots using a simple grid
3. Generate a working TTF font on your Desktop
